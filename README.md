# Splitwise Clone - Frontend

A React-based frontend for a Splitwise-like expense splitting application.

## Tech Stack

- **React 18** with TypeScript
- **Vite** - Build tool
- **React Router v6** - Routing
- **TailwindCSS v4** - Styling
- **shadcn/ui** - Component library
- **Tanstack Query** - Server state management
- **React Hook Form** + **Zod** - Form handling & validation
- **Axios** - HTTP client

## Features (Planned)

- User authentication via Google OAuth2
- Create and manage expense groups
- Add and split expenses with multiple split types
- View balances and settlement suggestions
- Responsive design for mobile and desktop

## Prerequisites

- Node.js 18 or higher
- pnpm (recommended) or npm
